Longest Increasing Subsequence

Veröffentlicht am 2016-08-19   |   in 工作

Given an unsorted array of integers, find the length of longest increasing subsequence.

For example,
Given [10, 9, 2, 5, 3, 7, 101, 18],
The longest increasing subsequence is [2, 3, 7, 101], therefore the length is 4. Note that there may be more than one LIS combination, it is only necessary for you to return the length.

题目出处：https://leetcode.com/problems/longest-increasing-subsequence/

思路参考：
http://www.cnblogs.com/dartagnan/archive/2011/08/29/2158247.html

方法1：用len[i]记录以输入数据nums[i]为最长上升子序列的终点时的最长上升子序列的长度。初始化len[i]为1，用以下公式更新len[i]。
len[i]=max{max{len[j]}+1,len[i]} 其中j满足[0,i）且nums[j] < nums[i]
